## Installing

setting .env (from .env.example)
<br>
Create database 'swapi' 
<br>
`composer install`
<br>
`php artisan migrate`
<br>
`php artisan swapi:parse` parsing data to database
<br>
`php artisan serve`
<br>
register on app

 - data on SWAPI users can not be viewed without registering and logging in. Viewing SWAPI data on route '/'.
